Security Analyst Position Requirements

Key skills and qualifications needed for the federal government department role

Job Overview

  • Position: Security Analyst
  • Location: Federal government department in Melbourne
  • Program: Enterprise data and analytics program
  • Methodology: Scaled agile delivery
  • Focus: Data visibility, automated processing, and information utilization

Required Skills and Qualifications

Risk Assessment Expertise

  • Proficiency in risk assessment methodologies
  • Experience in risk mitigation strategies
  • Knowledge of formal risk frameworks

Vulnerability Scanning

  • Ability to identify security vulnerabilities
  • Skills in proposing solutions based on software design
  • Experience with vulnerability management tools

API Development Security

  • Knowledge of security best practices for API development
  • Understanding of API security frameworks and protocols
  • Experience with secure API implementation

AWS Security Tools

  • Hands-on experience with AWS security services
  • Ability to review code for security issues
  • Cloud security implementation experience

Qualifications

  • Relevant cloud certifications
  • Data-related certifications
  • Technology certifications
  • Relevant tertiary education (already satisfied with Deakin University degree)

Additional Desired Attributes

  • Experience in large cloud and data projects
  • Ability to work in multi-disciplinary teams
  • Adaptability to scaled agile delivery methodology
  • Understanding of government security requirements
