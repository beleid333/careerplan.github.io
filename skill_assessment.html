Current Skills Assessment & Gap Analysis

Current Skills (Based on Information Technologies with Cybersecurity Major)

As a graduate of Information Technologies majoring in cybersecurity from Deakin University, you likely have the following foundational skills:

  1. General Cybersecurity Knowledge
  2. Basic security principles and concepts
  3. Understanding of common security threats and vulnerabilities
  4. Knowledge of security controls and countermeasures

  5. Familiarity with security frameworks and standards

  6. Network Security

  7. Understanding of network protocols and architectures
  8. Basic knowledge of firewalls, IDS/IPS, and network monitoring

  9. Experience with network traffic analysis

  10. Security Assessment

  11. Introduction to vulnerability assessment
  12. Basic penetration testing concepts

  13. Understanding of security auditing principles

  14. Cryptography

  15. Understanding of encryption algorithms and protocols
  16. Knowledge of cryptographic controls

  17. Basic implementation of cryptographic solutions

  18. Security Governance

  19. Introduction to security policies and procedures
  20. Understanding of compliance requirements
  21. Basic knowledge of risk management concepts

Skill Gaps for Security Analyst Position

Based on the job requirements, the following skill gaps need to be addressed:

  1. Risk Assessment Expertise
  2. Gap: Advanced risk assessment methodologies specific to government and enterprise environments
  3. Gap: Experience with formal risk frameworks (NIST, ISO 27005, etc.)

  4. Gap: Practical application of risk mitigation strategies

  5. Vulnerability Scanning

  6. Gap: Proficiency with enterprise vulnerability scanning tools
  7. Gap: Experience analyzing scan results in large-scale environments

  8. Gap: Ability to prioritize vulnerabilities based on business impact

  9. API Development Security

  10. Gap: Specific knowledge of API security best practices
  11. Gap: Understanding of OWASP API Security Top 10
  12. Gap: Experience with API gateway security configurations

  13. Gap: Secure API design principles

  14. AWS Security Tools

  15. Gap: Hands-on experience with AWS security services (GuardDuty, Security Hub, IAM, etc.)
  16. Gap: Cloud security architecture knowledge
  17. Gap: AWS security compliance frameworks

  18. Gap: Automated security scanning in AWS environments

  19. Certifications

  20. Gap: Industry-recognized cloud security certifications
  21. Gap: AWS-specific certifications

  22. Gap: Security analysis certifications

  23. Enterprise Experience

  24. Gap: Experience with large-scale data projects
  25. Gap: Familiarity with agile methodologies in security contexts
  26. Gap: Experience working in government security environments

About This Plan

This career development plan is designed for a graduate of Information Technologies with a major in cybersecurity from Deakin University, targeting a Security Analyst position at a federal government department in Melbourne, Australia.
